# 📚 Unidades Curriculares (UCs)

Bem-vindo à pasta **UCs**!

Aqui estão organizadas as **Unidades Curriculares** desenvolvidas durante o curso **Técnico em Informática do SENAC**. Cada pasta contém materiais, projetos, atividades, anotações e conteúdos relacionados à respectiva unidade curricular.

## 🎯 Objetivo

Este espaço foi criado para documentar minha evolução acadêmica e prática ao longo do curso, reunindo conhecimentos adquiridos em diferentes áreas da Tecnologia da Informação.

## 📂 Estrutura

As UCs estão separadas em pastas para facilitar a navegação e o acesso aos conteúdos de cada disciplina.

### Unidades Curriculares

- 🧠 **UC9 - Desenvolvimento de Algoritmos e Lógica de Programação**
  - Fundamentos da programação, algoritmos, pseudocódigo, fluxogramas e resolução de problemas.

> Novas UCs serão adicionadas conforme o andamento do curso.

## 🚀 Tecnologias e Conceitos Abordados

Ao longo das UCs, são trabalhados temas como:

- Hardware e Manutenção de Computadores;
- Sistemas Operacionais;
- Redes de Computadores;
- Lógica de Programação;
- Desenvolvimento de Software;
- Banco de Dados;
- Git e GitHub;
- Inteligência Artificial;
- Arduino e Automação.
