# 🧠 UC9 - Desenvolvimento de Algoritmos e Lógica de Programação

A **UC9**, ministrada pelo professor **Vinicius**, tem como objetivo desenvolver o raciocínio lógico e a capacidade de criar soluções computacionais por meio de algoritmos.

Durante esta unidade curricular, estou aprendendo a analisar problemas, planejar soluções e implementar algoritmos utilizando conceitos fundamentais da programação, tais como:

- 🔹 Variáveis e tipos de dados;
- 🔹 Operadores lógicos e matemáticos;
- 🔹 Estruturas condicionais (`if`, `else`, `switch`);
- 🔹 Estruturas de repetição (`for`, `while`, `do while`);
- 🔹 Fluxogramas e pseudocódigo;
- 🔹 Vetores e matrizes;
- 🔹 Boas práticas de programação.

## 🎯 Competências Desenvolvidas

- Pensamento lógico e analítico;
- Resolução de problemas computacionais;
- Desenvolvimento de algoritmos eficientes;
- Organização e estruturação de código;
- Base para o aprendizado de linguagens de programação.

## 🚀 Importância da UC9

Esta unidade curricular é uma das mais importantes do curso Técnico em Informática, pois fornece os fundamentos necessários para o desenvolvimento de softwares e para o aprendizado de diversas linguagens de programação.

> "A lógica é a base da programação. Antes de escrever código, é preciso aprender a pensar na solução."
>
